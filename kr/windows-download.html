다운로드 사용 설명서

참고 :

소프트웨어를 업그레이드할 때는 최신 버전의TourBox 콘솔만 설치하면 됩니다. 드라이브를 중복 설치할 필요가 없습니다.

macOS 시스템 사용하는 경우, 업그레이드하기 전에 소프트웨어가 V1.0번전인  경우 데이터 손실을 방지하기 위해 사전 설정 리스트를 미리 백업을 내보내십시오.

Windows 버전 설치 안내

TourBox를 컴퓨터에 연결하기 전에 최신 버전의 TourBox 콘솔 소프트웨어를 다운로드 하십시오.

드라이버 설치

1 단계: 사용중인 Windows 시스템 버전에 따라 해당 드라이버를 먼저 설치하십시오.

마우스 오른쪽 버튼을 클릭하여 "내 PC -- 속성" 을 열면 Windows 시스템 버전을 확인할 수 있습니다.

2 단계: 시스템 버전에 따라 해당 드라이버 파일의 압축을 푸십시오.

3 단계: 압축 푼 후 "exe. 파일"를 클릭하여 드라이버를 설치하십시오

콘솔 설치

최신 버전의 TourBox 콘솔 응용 프로그램을 설치하십시오.
