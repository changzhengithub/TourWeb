注意:

アプリケーションのアップグレードには、最新バージョンのTourBoxコンソールへ上書きアップグレードするだけで、ドライバの再インストールは不要です。

ご使用するシステムはmacOSで、アップグレード前のコンソールはV 1.0である場合、データの消失を防ぐために予め設定したプリセットをエクスポートし、バックアップしてください。

macOSバージョンのインストールガイド

TourBoxコンソールのインストール

1.1 「TourBox Consoel Setup.pkg」をアプリケーションにドラッグします。

インストール後、TourBoxコンソールを実行し、ポップアップウィンドウに従ってシステム設定を完了してください。

ポップアップウィンドウは、macOSシステムによって異なる場合があります。

2.1 TourBoxコンソールを実行して、システム環境設定を開き、TourBoxコンソールをmacOSで開くことができるようにしてください。

システム環境設定 -> セキュリティとプライバシー -> 一般 -> このまま開く

2.2 TourBoxコンソールを再度実行して、「開く」をクリックしてください。

2.3 ポップアップ時にシステムイベントを制御するアクセス要求を許可します。

2.4 アクセシビリティ設定を終了してください。

システム環境設定 -> セキュリティとプライバシー -> プライバシー -> アクセシビリティ -> TourBox Console

2.5 キーボード設定アシスタントを閉じてください。
