Installation Guide for Windows

Before connecting the TourBox to your computer, please download and install the Driver and the latest version of the TourBox Console Software.

Install the Driver

Step 1. Please follow the Windows system type to install the correct Driver.

You can check your Windows system type via right-click “This PC—Properties”.

Step 2. Please choose and unzip the Driver based on the Windows system type.

Step 3. Please clicking the “exe. file” to install the Drive.

Install TourBox Console

Please install the latest TourBox Console application.
