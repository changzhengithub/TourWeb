Installation Guide for macOS

Install the Application

1.1 Click the "TourBox Console Setup.pkg" file to install the TourBox Console.

After installation, please run the TourBox Console and follow the pop-up window to finish the system setting.

The pop-up windows may vary depends on different macOS system.

2.1 Please run the TourBox Console then open the System Preferences to allow macOS to open it.

System Preferences—>Security & Privacy—>General—>Open Anyway

2.2 Please run the TourBox Console again then click Open.

2.3 Allow the System Events accessing when it pop-up.

2.4 Please finish the Accessibility Setting.

System Preferences-->Security & Privacy-->Privacy-->Accessibility-->TourBox Console.

2.5 Please close the Keyboard Setup Assistant.
