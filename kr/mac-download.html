참고 :

소프트웨어를 업그레이드할 때는 최신 버전의TourBox 콘솔만 설치하면 됩니다. 드라이브를 중복 설치할 필요가 없습니다.

macOS 시스템 사용하는 경우, 업그레이드하기 전에 소프트웨어가 V1.0번전인  경우 데이터 손실을 방지하기 위해 사전 설정 리스트를 미리 백업을 내보내십시오.

macOS 버전 설치 안내

TourBox 콘솔 소프트웨어 설치

1.1 TourBox Consoel Setup.pkg을 ‘응용 프로그램’ 으로 드래그 하십시오.

설치 후, TourBox 콘솔 소프트웨어를 실행하고, 팝업창에 따라 시스템 설정을 완료하십시오.

macOS 시스템 버전에 따라 팝업창이 다를 수 있습니다.

2.1 TourBox 콘솔을 실행한 다음, 이를 시스템 환경설정에 macOS가 열 수 있도록 설치하십시오.

시스템 환경설정 —> 보안 및 개인 정보 보호 —> 일반 —> 확인 없이 열기

2.2 TourBox 콘솔을 다시 실행 한 다음, 열기를 클릭하십시오.

2.3 ‘System Events’ 를 제어 할 수 있도록 팝업창에 ‘확인’버튼을 클릭하십시오.

2.4 손쉬운 사용 설정을 완료하십시오.

시스템 환경설정-> 보안 및 개인 정보 보호-> 개인 정보 보호-> 손쉬운 사용 -> TourBox Console

2.5 키보드 설정 지원 창을 닫으십시오.
